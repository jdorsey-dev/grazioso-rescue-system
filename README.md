# Grazioso Salvare Rescue System

## Description
This is a Java-based application developed for Grazioso Salvare, a search-and-rescue animal training company. The program manages rescue animals by storing and organizing their information using object-oriented programming principles.

## Features
- Stores and manages rescue animal data
- Supports multiple animal types (e.g., dogs and monkeys)
- Uses class inheritance to extend functionality
- Allows intake of new animals
- Displays and searches for available rescue animals

## Skills Used
- Java
- Object-Oriented Programming (OOP)
- Inheritance and encapsulation
- Classes and methods
- ArrayLists for data management

## What I Learned
- How to design and implement classes in Java
- How inheritance allows code reuse and extension
- How to organize larger programs using multiple classes
- How to manage and manipulate collections of objects

## How to Run
Compile and run the program in a Java environment (such as VS Code or an IDE like Eclipse). The main driver class controls the program flow and user interaction.
